## 自己紹介ページの作成
サークルCode.gate?での課題である自己紹介ページを作成する

## 概要
- 自己紹介ページ
- ブログ機能
- Github,Xのリンク作成

## インストール
### 自分の実行環境
- node 24.8.0
- npm 11.6.0

### インストール手順
1. リポジトリをクローン
``` bash
git clone https://github.com/yuji0276/portfolio.git
```
2. ディレクトリ移動
``` bash
cd portfolio
```
3. 依存関係をインストール
``` bash
npm install
```
4. アプリケーションを起動
``` bash
npm run dev
```
